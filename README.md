Fashion-MNIST Classification using MLP 👕👟This repository contains a Neural Networks project that implements a Multi-Layer Perceptron (MLP) using Keras/TensorFlow to classify clothing items and accessories from the popular Fashion-MNIST dataset.🛠️ Project PipelineData Loading: Imports 60,000 training images and 10,000 test images (each $28 \times 28$ pixels in grayscale).Preprocessing:Normalization: Scales pixel values between 0 and 1 by dividing by 255.0.One-Hot Encoding: Converts target labels into categorical format for multi-class classification.Data Splitting: Splits the training data to set aside 20% for validation.Model Architecture:Flatten layer to convert the $28 \times 28$ image matrix into a 1D vector.Dense hidden layer (128 neurons, ReLU activation).Dropout layer (20%) to prevent overfitting.Dense hidden layer (64 neurons, ReLU activation).Dense output layer (10 neurons, Softmax activation) representing the 10 fashion classes.Compilation & Training: Uses the Adam optimizer and Categorical Crossentropy loss function. The model is trained for 15 epochs with a batch size of 32.Evaluation & Visualization: Evaluates performance on test data and plots curves for both loss and accuracy.🚀 RequirementsTo run this code locally, ensure you have the following libraries installed:Bashpip install numpy matplotlib tensorflow scikit-learn
📊 Expected OutputUpon running the script, the project will:Display a grid of 9 sample images with their respective class titles.Print the architecture summary using model.summary().Output the final Test Loss and Test Accuracy after 15 epochs.Generate visual plots comparing Training vs. Validation for both Loss and Accuracy.
